For en JS

La estrctura For dentro de JS nos permite trabjar tareas que puedan ser repetidas, pudiendo ahorrar lineas de codigo y realizar principalmente busquedas dentro de contenidos como arreglos

Estrcutura

el for contara con la siguiente estructura : 

			for(inicializacion; condicion; actualizacion) {
 			 ...
			}
En donde :
  • La "inicialización" es la zona en la que se establece los valores iniciales de las variables que controlan la repetición.
  • La "condición" es el único elemento que decide si continua o se detiene la repetición.
  • La "actualización" es el nuevo valor que se asigna después de cada repetición a las variables que controlan la repetición.
