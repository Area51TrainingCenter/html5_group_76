Selectores en JS

Los selectores nos van a permitir tener una referencia de los elemntos html para poder interactuar en JS, a esto se le conoce como manejo DOM

Tipo de Selectores en JS
  • GetElementById
  • GetElementsByTagName
  • GetElementsByClassName
  • querySelectorAll

GetElementById

uso : document.getElementById()

Devuelve : Un solo elemento

GetElementsByTagName

uso : document.getElementsByTagName("p")

Devuelve : Un arreglo elementos

GetElementsByClassName

uso : document.getElementsByClassName("miclase")

Devuelve : Un arreglo elementos

querySelectorAll

uso : document.querySelector('#ejemplo1 li');

Devuelve : Un arreglo elementos
